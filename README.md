🍻 App Web de Micheladas
Esta es una aplicación web desarrollada con Python y Flask para gestionar pedidos de micheladas de manera sencilla y eficiente.

✨ Características
Registro de clientes 📋
Gestión de pedidos 🛒 (productos, precios y cálculo de cambio)
Eliminación automática de pedidos pagados ✅
Sistema de autenticación 🔒 (inicio de sesión para controlar el acceso)
🛠 Tecnologías utilizadas
Backend: Python + Flask
Base de datos: SQLite/MySQL (según configuración)
Frontend: HTML, CSS, Tailwindcss
